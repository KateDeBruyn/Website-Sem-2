Data-Driven Art

The Keyboard Graph

The Keyboard Graph is inspired by Giorgia Lupi's data-driven artworks that include static images of real life with superimposed data visualisation. The visualisation techniques are illustrated to build 'themselves' from the forms found in the real life photographs Lupi uses.

The Keyboard Graph's data is based off of the data used in visualisations found here. An API used to track listings of game's deal ratings, Steam ratings, and prices was used for this site's data. More information on the making of this artwork can found in the blog, Data-Driven Art Reflection.

HTML and CSS based on keightyrose's previous project. Available at: https://github.com/KateDeBruyn/KateDeBruyn.github.io
